# TestRestaurant Geneva - Design System

## Visual Direction
Elegant Sicilian luxury with Sicilian warmth - combining sophistication with authentic family hospitality.

## Color Palette
**Primary Colors:**
- Black: `#000000` - Main text, headers, sophisticated backgrounds
- Gold: `#D4AF37` - Accents, CTAs, luxury highlights
- Rich Gold: `#B8860B` - Hover states, secondary accents

**Supporting Colors:**
- Warm Cream: `#FFF8E7` - Light backgrounds, cards
- Dark Charcoal: `#1A1A1A` - Sections, contrast areas
- Soft Gold: `#F4E4C1` - Subtle highlights, borders
- White: `#FFFFFF` - Clean sections, text on dark

## Typography
**Primary Font:** Playfair Display (serif) - Elegant headers, luxury feel
**Secondary Font:** Inter (sans-serif) - Body text, readability

**Scale:**
- Hero: 3.5rem / 56px
- H1: 2.5rem / 40px
- H2: 2rem / 32px
- H3: 1.5rem / 24px
- Body: 1rem / 16px
- Small: 0.875rem / 14px

## Layout Principles
1. **Asymmetric Hero:** Large image with overlaid text, gold accent lines
2. **Section Rhythm:** Alternating dark/light backgrounds for visual interest
3. **Generous Spacing:** Breathing room between sections (80-120px)
4. **Image-First:** Large, high-quality photos showcasing food and ambiance
5. **Gold Dividers:** Thin gold lines to separate sections elegantly

## Component Style
- **Buttons:** Black background, gold text, gold border on hover
- **Cards:** Cream background, subtle shadow, gold accent on hover
- **Forms:** Clean inputs with gold focus states
- **Navigation:** Fixed top nav, black background, gold highlights

## Responsive Approach
- Mobile-first design
- Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- Stack sections vertically on mobile
- Maintain generous spacing on all devices

## Photography Treatment
- Full-width hero images
- Grid layouts for galleries (3 columns desktop, 1-2 mobile)
- Subtle overlay on text-over-image sections
- High contrast for readability
